# MESSAGING-APP
Simple Messaging app using Node JS, Express And MongoDB

### Prerequisites
```
(1) NodeJs
(2) MongoDb
(3) Express
```


## API Paths

### (1) POST /login
```
uname : String
password : String

```
### (2) POST /register

```
uname: String
fname: String
lname: String
password: String
confirm_password: String
```

### (3) POST /sendmessage

```
recipient : String
subject : String
body : String

```
